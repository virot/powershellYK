# Reset-YubikeyOATH

## SYNOPSIS
Reset the entire YubiKey OATH application.

## SYNTAX

```
Reset-YubikeyOATH [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Reset the entire Yubikey OATH application.

## EXAMPLES

### Example 1
```powershell
PS C:\> Reset-YubikeyOATH
```

Resets the entire Yubikey OATH application.
