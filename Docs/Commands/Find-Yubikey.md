# Find-Yubikey

## SYNOPSIS

Lists all YubiKeys on system

## SYNTAX

### Default (Default)

```
Find-Yubikey [-OnlyOne] [-Serialnumber <Int32>] [<CommonParameters>]
```

### __AllParameterSets

```
Find-YubiKey [-OnlyOne] [-Serialnumber <int>] [<CommonParameters>]
```

## ALIASES

## DESCRIPTION

List all YubiKeys on system

## EXAMPLES

```powershell
PS C:\> Find-Yubikey
```
