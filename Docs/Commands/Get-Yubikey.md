# Get-Yubikey

## SYNOPSIS
Returns the connected Yubikey

## SYNTAX

```
Get-Yubikey [<CommonParameters>]
```

## DESCRIPTION
This command will return information about the currently connected Yubikey

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-Yubikey
```

Information about Yubikey
