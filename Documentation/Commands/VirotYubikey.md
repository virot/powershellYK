# VirotYubikey Module
## Description
A powershell module that use the Yubikey SDK to allow interaction with yubikeys

## VirotYubikey Cmdlets
### [Block-YubikeyPIV](Block-YubikeyPIV.md)
Allows blocking parts of the YubikeyPIV

### [Connect-Yubikey](Connect-Yubikey.md)
Connect the yubikey to the module

### [Connect-YubikeyFIDO2](Connect-YubikeyFIDO2.md)
Connect to the Yubikey FIDO2 module

### [Connect-YubikeyPIV](Connect-YubikeyPIV.md)
Connect to the Yubikey PIV module

### [Disconnect-Yubikey](Disconnect-Yubikey.md)
Disconnect from the Yubikey

### [Export-YubikeyPIVCertificate](Export-YubikeyPIVCertificate.md)
Export a certificate from the Yubikey

### [Find-Yubikey](Find-Yubikey.md)
Search for a Yubikey

### [Get-Yubikey](Get-Yubikey.md)
Get information from the connected Yubikey

### [Get-YubikeyPIV](Get-YubikeyPIV.md)
Get information from the connected Yubikey PIV module

### [Import-YubikeyPIVCertificate](Import-YubikeyPIVCertificate.md)
Import certificates to the Yubikey PIV

### [New-YubikeyPIVCSR](New-YubikeyPIVCSR.md)
Genreate a new CSR for a slot in the Yubikey PIV

### [New-YubikeyPIVKey](New-YubikeyPIVKey.md)
Genreate a new Key for a slot in the Yubikey PIV

### [New-YubikeyPIVSelfSign](New-YubikeyPIVSelfSign.md)
Genreate a new Selfsigned Certificate for a slot in the Yubikey PIV

### [Reset-YubikeyPIV](Reset-YubikeyPIV.md)
Rest the PIV part of the Yubikey

### [Set-YubikeyPIV](Set-YubikeyPIV.md)
Set values in the PIV module of the Yubikey
