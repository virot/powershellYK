# Disconnect-Yubikey

## SYNOPSIS
Disconnects the Yubikey

## SYNTAX

```
Disconnect-Yubikey [<CommonParameters>]
```

## DESCRIPTION
Disconnect Yubikey, to allow other processes to connect

## EXAMPLES

### Example 1
```powershell
PS C:\> Disconnect-Yubikey
```

Disconnects the Yubikey
